### 项目内容：

本项目包含Quantumult X的基本配置文件及常用脚本（破解脚本仅供学习交流使用）

由于Quantumlt X从1.0.3版本之后不再支持远程脚本挂载 所以本项目只提供WorkingCopy本地挂载的配置文件

Rules文件夹内包含分流规则及脚本配置文件 开箱即用

JS则为脚本文件 脚本由GitHub大佬制作 此处仅为优质脚本整理 （仅供学习交流使用）

本仓库为自己使用，长期维护，不提供技术支持。
